## Run Locally

Clone the project

```bash
  git clone https://github.com/dr0id007/react-native-boilerplate.git
```

Go to the project directory

```bash
  cd react-native-boilerplate
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run android
  npm run ios
```
